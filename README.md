# Coffeeroasters Website

A responsive multi-page coffee subscription website built using HTML, CSS, and JavaScript.

## Live Demo

https://baxti-coffe-roasters.netlify.app/

## Project Purpose

This project coffee subscription service website.
It was built to demonstrate my skills in responsive web design, clean HTML structure,
and scalable CSS architecture using the BEM methodology.

## Features

- Responsive layout for desktop, tablet, and mobile
- Multi-page navigation (Home, About, Create Your Plan)
- Coffee subscription plan configurator
- Mobile hamburger menu
- Reusable and modular UI components
- BEM-based CSS structure

## Technologies Used

- HTML5 (semantic markup)
- CSS3 (Flexbox, Media Queries)
- BEM methodology
- Vanilla JavaScript

## Folder Structure

- css/
- js/
- img/
- fonts/
- index.html
- about.html
- create-your-plan.html

## What I Learned

- Building scalable CSS using BEM
- Solving responsive layout issues with decorative images
- Creating reusable components
- Improving UX for mobile navigation
